# AI Tooling

Notes about AI tooling used while developing PraxaApp:

- I used `langchain` and `langchain-community` packages for model and retrieval orchestration.
- `sentence-transformers` via `langchain-huggingface` for embeddings.
- `Chroma` local vector DB via `langchain-chroma` for persistence.
- Development used iterative prompting and small automation to generate sample policies.
