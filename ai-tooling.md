# AI Tooling

This project was developed with a practical AI engineering workflow that combined general coding tools with retrieval and model orchestration libraries.

- GitHub Copilot was used to speed up implementation, boilerplate generation, and documentation updates.
- LangChain and langchain-community were used for model integration, message handling, and retrieval orchestration.
- Hugging Face sentence-transformers and langchain-huggingface were used for embeddings.
- Chroma was used as the local vector database for persistent document indexing.
- The project also uses iterative prompting and small test-driven refinements to improve answer quality, citation formatting, and UI behavior.
